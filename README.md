# ✦ 10 Stylish Infinite Logo / Partner Sliders

A standalone HTML/CSS showcase of **10 different marquee-style sliders**.  
Each slider has a completely distinct visual design — from 3D perspective and glassmorphism to retro pixel and morphing blobs.

Perfect for partner/sponsor logos, brand carousels, or any modern website.

## 🎨 Slider Styles Included
1. **Round Logo** – Gold accent, warm glow  
2. **3D Perspective Depth** – Angled cards with hover lift  
3. **Neon Cyberpunk Glow** – Dark theme, glowing borders  
4. **Glassmorphism Frosted** – Backdrop blur, translucent cards  
5. **Minimalist Elegant Line** – Thin circular borders, subtle dot  
6. **Bento Grid Modern** – Rounded cards with rotating gradient  
7. **Retro Pixel 8‑bit** – Pixelated borders and blocky shadows  
8. **Morphing Blob Organic** – Continuously changing border radius  
9. **Flip Card Reveal** – 3D flip to show back content  
10. **Elastic Stretch Bouncy** – Exaggerated scale on hover  

## 🚀 How to Use
1. Clone the repository or download `index.html`.  
2. Open it in any modern browser – no build step, no dependencies.  
3. Replace the placeholder text/emojis with your own logos or images.  
4. (Optional) Host on GitHub Pages, Netlify, or your own server.

## 🔧 Customisation
- **Speed**: change `animation-duration` inside the `@keyframes` or the CSS class.  
- **Images**: replace the inner HTML of each slider card with `<img>` tags.  
- **Colours**: modify the CSS variables or inline styles.
